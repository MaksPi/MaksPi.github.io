Задание на покраску

КОМПАС-групп Для сотрудников Отделы компании Конструкторский отдел Рабочие инструкции Задание на покраску
  1. Создать чертеж, на котором:
    1. Изобразить сечения всех профилей, которые предусмотрены к покраске.
    2. Изобразить окрашиваемую поверхность на каждом сечении профиля.
    3. Каждое сечение профиля промаркировать артикульным номером, согласно каталога применяемой профильной системы.
  2. Составить спецификацию, в которой:
    1. Указать артикульные номера всех профилей и всех элементов фурнитуры, которые предусмотрены к покраске.
    2. Указать названия всех артикульных номеров.
    3. Указать длины всех профилей.
    4. Указать количество всех длин каждого профиля и всех элементов фурнитуры, которые предусмотрены к покраске.

Исходными данными для получения списка материалов, которые предусмотрены к покраске, являются:

  1. Подтвержденный нами счёт от поставщика.
  2. Наша заявка, которая для поставщика является основанием для выставления счёта.
