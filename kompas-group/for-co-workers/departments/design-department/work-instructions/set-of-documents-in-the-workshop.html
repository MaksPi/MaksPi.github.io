КД (комплект документов) в цех

КОМПАС-групп Для сотрудников Отделы компании Конструкторский отдел Рабочие инструкции КД в цех

Общие правила, подходящие для всех типов (видов) конструкций:

  1. Системные файлы (.LOB, .PON и т.д.) с откорректрованными конфигурацями и размерами конструкций должны быть сохранены по следующему адресу на сервере:

    2. \\Nasc4fa3c\kompas\КОММЕРЧЕСКИЕ ПРЕДЛОЖЕНИЯ\ГОД ЗАКАЗА\ЗАКАЗ\В работу\В цех

  2. Для удобства определения сотрудниками цеха цвета материалов, на передаваемых документах в цех, рядом с указанием цвета согласно цветового стандарта RAL, указывать данный цвет в упрощенной форме - словом.
  3. Перед передачей КД в цех проверить, чтобы состав и количество всех материалов в спецификации, передаваемой в цех, соответствовали составу и количеству всех материалов в счёте.
  4. Если в конструкциях стеклопакеты со шпросами, то КД в цех отдавать только после готовности специфификации и чертежей этих стеклопакетов.
  5. В чертежах стеклопакетов с размещением шпросов обязательно предусмотреть взаимное расположение шпросов таким образом, чтобы все горизонтальные шпросы были размещены на общей отметке здания, а все вертикальные шпросы - по единой оси.

Окна / двери / гармошки / раздвижки:

  1. "Сборочная спецификация" (Reynaers) / "Производственный лист" (Ponzio) либо подобный документ другой системы, полученный из программы (либо созданный "вручную"), изготавливаемых конструкций.
    1. В случае применения встраиваемого привода автоматического открывания либо встраиваемого автоматического доводчика, вместе со сборочными спецификациями (либо другими подобными документами) в цех необходимо передать чертежи всех необходимых фрезеровок, сверловок и т.д. для всех профилей, на которые будут устанавливаться данные привода либо доводчики.
  2. Горизонтальные и вертикальные сечения всех изготавливаемых конструкций по всем характерным местам в хорошо читаемом масштабе.
  3. Оптимизация профилей, полученная из программы (либо созданная "вручную"), изготавливаемых конструкций.
  4. Спецификация материалов, изготавливаемых конструкций.
  5. Для раздвижек: проверить, чтобы ширины створок были подобраны таким образом, чтобы глухие створки не препядствовали подвижным створкам полностью открыться в световом проёме.

Фасадка:

  1. "Сборочная спецификация" (Reynaers) / "Производственный лист" (Ponzio) либо подобный документ другой системы, полученный из программы (либо созданный "вручную"), изготавливаемых конструкций.
  2. Схема размещения стоек и ригелей с привязкой размещения ригелей от единого общего базового конца стойки.
    1. В примечании данной схемы должны быть ссылки на необходимые страницы каталога, на которых представлены способы обработки тех или иных элементов конструкции.
  3. Горизонтальные и вертикальные сечения всех изготавливаемых конструкций по всем характерным местам в хорошо читаемом масштабе.
  4. Спецификация ригелей.
  5. Оптимизация профилей, полученная из программы (либо созданная "вручную"), изготавливаемых конструкций.
  6. Спецификация материалов, изготавливаемых конструкций.

Крыша:

  1. Схема размещения элементов крыши.
  2. Поперечный разрез по крыше.
  3. Продольный разрез по крыше.
  4. Оптимизация профилей, созданная "вручную", изготавливаемой конструкции.
  5. Чертежи всех необходимых фрезеровок, сверловок и терморезов для всех элементов, изготавливаемой конструкции.
  6. Спецификация материалов, изготавливаемой конструкции.
