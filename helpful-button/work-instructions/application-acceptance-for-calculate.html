Прием заявки для первичного расчета:

Исходные данные заявки на расчет должны в себе содержать:

  1. Адрес объекта.
  2. Этаж / этажи на которых находятся конструкции.
  3. Эскизы конструкций с размерами и описанием всех необходимых конструктивных особенностей:
    1. Для всех видов конструкций:
      1. Профильная система.
      2. Система фурнитуры.
      3. Цвет профилей.
      4. Цвет фурнитуры.
      5. Формулы стеклопакетов или описания других видов заполнения.
      6. Ориентация конструкции (вид снаружи / вид изнутри).
      7. Условия примыкания / крепления конструкций к зданию или между собой, на которые нужно обратить внимание.
      8. Количество каждой из конструкций.
    2. Окна:
      1. Тип / вид открывания.
      2. Способ открывания / закрывания (детский замок, привод автоматического открывания, ограничитель открывания и т.д.).
      3. Тип фурнитуры (видимые петли / скрытые петли).
    3. Двери:
      1. Ориентация открываний (наружу / вовнутрь).
      2. Указать активную и штульповую створки.
      3. Тип петель (лепестковые / роликовые / скрытые).
      4. Тип / вид ручки (нажимная / офисная).
      5. Расположение ручки по высоте.
      6. Тип замка (1-но точечный / 3-х точечный / 7-ми точечный).
      7. Способ открывания / закрывания (ключ - ключ, ключ - ручка-барашек, магнитный замок, автоматический доводчик и т.д.).
      8. Длина личины (целая или половинчатая).
      9. Система антипаники (есть / нет).
    4. Раздвижки:
      1. Ориентация открываний (влево / вправо).
      2. Указать активную и пасивную створки.
      3. Тип / вид ручек (ручка-рычаг / ручка-ракушка).
      4. Расположение ручек по высоте.
