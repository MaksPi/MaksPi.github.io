Изменение документа

Модификации DOM – это ключ к созданию «живых» страниц.

Можно создавать новые элементы «на лету» и изменять уже существующие.

Пример: показать сообщение

Методы на примере – а именно, добавление на странице сообщения, которое будет выглядеть получше, чем alert.

<style> .alert { padding: 15px; border: 1px solid #d6e9c6; border-radius: 4px; color: #3c763d; background-color: #dff0d8; } </style> <div class="alert"> <strong>Всем привет!</strong> Вы прочитали важное сообщение. </div> 1
