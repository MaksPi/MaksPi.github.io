Переменные

Переменная – это «именованное хранилище» для данных.

Для создания переменной в JavaScript, используется ключевое слово let.

Приведённая ниже инструкция создаёт (другими словами: объявляет или определяет) переменную и помещает в переменную данные, используя оператор присваивания =:

let message = 'Hello!'; // объявляем переменную и присваиваем ей значение

alert(message); // Hello! 1

Можно объявить несколько переменных в одной строке:

let user = 'John!', age = 25, message = 'Hello!'; 1

Но для лучшей читаемости кода лучше объявлять каждую переменную на новой строке:

let user = 'John!',
age = 25,
message = 'Hello!'; 1

var вместо let

Устаревшее ключевое слово var - почти то же самое, что и let:

var message = 'Hello!'; 1

Оно тоже объявляет переменную, но немного по-другому, «устаревшим» способом.

На первый взгляд, поведение var похоже на let. Например, объявление переменной:

function sayHi() {
var phrase = "Привет!"; // локальная переменная, "var" вместо "let"

alert(phrase); // Привет
}

sayHi();

alert(phrase); // Ошибка: phrase не определена

Для «var» не существует блочной области видимости

Область видимости переменных var ограничивается либо функцией, либо, если переменная глобальная, то скриптом. Такие переменные доступны за пределами блока.

Например:

if (true) {
var test = true; // используем var вместо let
}

alert(test); // true, переменная существует вне блока if 1

Имена переменных

В JavaScript есть два ограничения, касающиеся имён переменных:

  1. Имя переменной должно содержать только буквы, цифры или символы $ и _.
  2. Первый символ не должен быть цифрой.

Примеры допустимых имен:

let userName;
let test123; 1
2

Если имя содержит несколько слов, обычно используется верблюжья нотация, то есть, слова следуют одно за другим, где каждое следующее слово начинается с заглавной буквы: myVeryLongName.

Регистр имеет значение

Переменные с именами apple and AppLE – это две разные переменные.

Зарезервированные имена

Существует список зарезервированных слов, которые нельзя использовать в качестве имён переменных, потому что они используются самим языком.

Например: let, class, return и function зарезервированы.

Приведённый ниже код даёт синтаксическую ошибку:

let let = 5; // нельзя назвать переменную "let", ошибка!
let let = 5; // также нельзя назвать переменную "return", ошибка! 1
2

Константы

Чтобы объявить константную, то есть, неизменяемую переменную, используется const вместо let:

const myBirthday = '08.06.1984'; 1

Переменные, объявленные с помощью const, называются «константами». Их нельзя изменить. Попытка сделать это приведёт к ошибке.

Константы в верхнем регистре

Широко распространена практика использования констант в качестве псевдонимов для трудно запоминаемых значений, которые известны до начала исполнения скрипта.

Названия таких констант пишутся с использованием заглавных букв и подчёркивания.

Например, константы для различных цветов в «шестнадцатеричном формате»:

const COLOR_RED = "#F00";
const COLOR_GREEN = "#0F0";
const COLOR_BLUE = "#00F";
const COLOR_ORANGE = "#FF7F00";

// ...когда нам нужно выбрать цвет
let color = COLOR_ORANGE;
alert(color); // #FF7F00 1
