КД (комплект документов) на монтаж

КОМПАС-групп Для сотрудников Отделы компании Конструкторский отдел Рабочие инструкции КД на монтаж

Окна / двери / гармошки / раздвижки:

  1. Рабочий проект, который в себя должен включать:
    1. Схема размещения конструкций на планах, разрезах и фасадах здания по необходимости.
    2. Узлы примыкания конструкций к зданию.
  2. Чертежи деталей примыканий.
  3. Чертежи стадии КМ.
  4. Чертежи стадии КМД.
  5. Чертежи КМД закладных деталей.
  6. Ведомость конструкций.
  7. Спецификация "дополнительных" материалов.
  8. Спецификация заполнений.

Фасадка:

  1. Рабочий проект, который в себя должен включать:
    1. Схема размещения конструкций на планах, разрезах и фасадах здания по необходимости.
    2. Узлы примыкания конструкций к зданию.
    3. Конструктивная схема с указанием "несущих" и "опорных" кронштейнов в конструкции.
    4. Схема размещения стоек и ригелей.
    5. Схема размещения заполнений.
  2. Чертежи деталей примыканий.
  3. Чертежи КМД закладных деталей.
  4. Ведомость конструкций.
  5. Спецификация "дополнительных" материалов.
  6. Спецификация заполнений.

Крыша:

  1. Рабочий проект, который в себя должен включать:
    1. Схема размещения стоек с боковыми конструкциями (или без боковых конструкций) на плане (по необходимости).
    2. Схема размещения элементов крыши.
    3. Схема размещения заполнений.
    4. Узлы примыкания конструкций между собой.
    5. Узлы примыкания конструкций к зданию.
  2. Чертежи деталей примыканий.
  3. Чертежи стадии КМ.
  4. Чертежи стадии КМД.
  5. Чертежи КМД закладных деталей.
  6. Спецификация "дополнительных" материалов.
  7. Спецификация заполнений.
